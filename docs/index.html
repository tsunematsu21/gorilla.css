Layouts

Container

The container to center content horizontally, like the this documentation, provides visibility for narrow angle gorilla.

<div class="container">
  <!-- Your awesome content -->
</div>

Section

The section is a container to divide page into sections, like the this documentation.
Add space between sections and between title and body to improve readability for any gorilla.

<div class="container">
  <div class="section">
    <h1>Section 1 title</h1>
    <p>Section 1 body</p>
  </div>
  <div class="section">
    <h1>Section 2 title</h1>
    <p>Section 2 body</p>
  </div>
</div>

Button

The button is an essential interface element for any user experience.
A shadow transition when press the button, provides gorilla-friendly interaction.



<button class="button">.button</button>

Also, can be disabled to prevent press the button by gorilla.



<button class="button" disabled>.button</button>

Color variant

Coloring will improve visibility.
Inside the button, text color is calculated according to background color.



<button class="button is-primary">.is-primary</button>
<button class="button is-secondary">.is-secondary</button>

For fashionable gorilla, a utility class that enforces text color is also provided.



<button class="button is-primary has-text-white">.has-text-white</button>
